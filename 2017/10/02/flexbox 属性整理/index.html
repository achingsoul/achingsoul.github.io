Flexbox 属性整理

发表于 2017-10-02 | 分类于 CSS

以前收藏的两个网站，对于学习 Flexbox 比较轻松易懂，多玩几次就会了，顺便整理一下属性和取值，方便自己复习。

网站一：http://flexboxfroggy.com/#zh-cn 网站二：http://www.flexboxdefense.com/

Flexbox属性整理

注意当flex以列为方向时，justify-content 控制纵向对齐，align-items 控制横向对齐。

justify-content 水平对齐元素

  • flex-start: 元素和容器的左端对齐。
  • flex-end: 元素和容器的右端对齐。
  • center: 元素在容器里居中。
  • space-between:元素之间保持相等的距离。
  • space-around:元素周围保持相等的距离。

align-items 纵向对齐元素

align-self 控制单个元素属性

  • flex-start: 元素与容器的顶部对齐。
  • flex-end: 元素与容器的底部对齐。
  • center: 元素纵向居中。
  • baseline: 元素在容器的基线位置显示。
  • stretch: 元素被拉伸以填满整个容器。

align-content

当作为一个弹性盒子容器的属性时，浏览器如何在容器的侧轴围绕弹性盒子项目分配空间。
  • flex-start: 多行都集中在顶部。
  • flex-end: 多行都集中在底部。
  • center: 多行居中。
  • space-between: 行与行之间保持相等距离。
  • space-around: 每行的周围保持相等距离。
  • stretch: 每一行都被拉伸以填满容器。

flex-direction 元素在容器里摆放的方向

  • row: 元素摆放的方向和文字方向一致。
  • row-reverse: 元素摆放的方向和文字方向相反。
  • column: 元素从上放到下。
  • column-reverse: 元素从下放到上。

flex-wrap 指定元素单行or多行显示

  • nowrap: 所有的元素都在一行。
  • wrap: 元素自动换成多行。
  • wrap-reverse: 元素自动换成逆序的多行。

flex-flow 上面两个属性的缩写，接受两个属性的值，以空格隔开

  • 例：flex-flow: row wrap

order 调整元素的顺序，默认值为0
